# Sistema de Gestión de Inventario

## Resumen del Proyecto

Este proyecto integra los conocimientos adquiridos durante la primera fase del plan de desarrollo full stack, 
combinando Spring Boot como backend, MySQL como base de datos y Angular para el frontend. La aplicación permite 
gestionar productos, categorías, proveedores y movimientos de inventario con un sistema CRUD completo.

## Objetivos de Aprendizaje

- Implementar un backend completo con Spring Boot
- Diseñar y desarrollar un modelo de datos relacional en MySQL
- Crear una interfaz de usuario con componentes reutilizables en Angular
- Integrar los sistemas mediante APIs RESTful
- Aplicar validaciones y manejo de errores
