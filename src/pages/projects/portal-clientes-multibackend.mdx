# Portal de Clientes Multibackend

## Resumen del Proyecto

Este proyecto implementa un portal web completo para clientes, integrando múltiples backends y tecnologías aprendidas durante la segunda fase. La aplicación combina microservicios en Java/Spring y .NET Core unidos mediante un API Gateway, y presenta un frontend moderno en Angular con Material Design y gráficos interactivos, todo ello contenerizado con Docker.

## Objetivos de Aprendizaje

- Diseñar e implementar microservicios con diferentes tecnologías (.NET Core y Spring Boot)
- Crear una capa de integración mediante API Gateway
- Implementar autenticación y autorización en arquitecturas distribuidas
- Desarrollar interfaces avanzadas con Angular Material y visualización de datos
- Introducir contenedores Docker para estandarizar entornos de desarrollo y despliegue
