# Sistema de Gestión de Tickets Seguro

## Resumen del Proyecto

Este proyecto implementa una plataforma empresarial completa para la gestión de tickets de soporte e incidencias con alto nivel de seguridad. La aplicación se desarrolla con arquitectura de microservicios usando Spring Cloud, autenticación centralizada con OAuth2/JWT, y se despliega en contenedores orquestados con Rancher sobre RedHat Linux mediante pipelines automatizados de CI/CD.

## Objetivos de Aprendizaje

- Implementar arquitecturas avanzadas de microservicios con Spring Cloud
- Aplicar estándares de seguridad empresarial (OAuth2, JWT, RBAC)
- Configurar orquestación de contenedores con Rancher
- Desarrollar pipelines de CI/CD completos con Jenkins
- Implementar protecciones contra vulnerabilidades comunes (OWASP Top 10)
